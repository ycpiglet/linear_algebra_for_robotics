목표 경로: 칼만 필터(Kalman filter)를 직관에서 완전 유도까지

칼만 필터는 한 번에 이해하지 않아도 된다

칼만 필터(Kalman filter)는 선형대수·확률·동역학이 합류하므로 처음부터 모든 유도를 밀어붙이면 어느 축에서 막혔는지 알기 어렵다. 아래 경로는 작동 감각 → 계산 → 유도 → 증명을 분리한다.

1. 직관 경로 — 약 70분

평균·분산 → 상태와 측정 → 두 추정의 가중 결합 → 예측/보정
  1. 평균과 공분산의 30초 핵심과 스칼라 분산을 읽는다.
  2. 상태공간의 등속도(constant-velocity) 예를 읽는다.
  3. 칼만 필터의 스칼라 예를 손으로 계산한다.

이 단계에서는 확률과정, 슈어 여원(Schur complement), 행렬미적분(matrix calculus)이 필요하지 않다.

2. 구현 경로 — 약 3시간

  1. \(F,H,Q,R,P\)의 모양(shape)과 단위를 표로 작성한다.
  2. 예측(prediction)과 보정(correction)을 한 단계 계산한다.
  3. 역행렬을 직접 만들지 않고 선형계 풀이(solve)를 사용하며, 조지프 형식으로 공분산을 갱신한다.
  4. 측정 결측(measurement dropout)에서는 보정을 건너뛰고 예측만 수행한다.
  5. 혁신(innovation)과 정규화 혁신제곱(NIS)을 기록한다.
  6. Go2·L2 교육용 수치 사례에서 교차공분산이 속도와 편향을 함께 고치는 과정을 확인한다.

3. 유도 경로 — 약 6시간

공동 가우스 → 조건부 분포 → 칼만 이득
       └→ 투영/가중최소제곱 → 같은 보정
  1. 가우스 조건부 분포(Gaussian conditioning)를 유도한다.
  2. 행렬 보정의 가우스 조건부 유도로 복귀한다.
  3. 최소제곱의 가중최소제곱과 직교성 관점을 읽는다.
  4. 가우스 조건부 분포와 가중최소제곱이 같은 칼만 이득에 도달함을 설명한다.

4. 증명·진단 경로 — 9시간 이상

  1. 직교투영에서 잔차의 직교성을 증명한다.
  2. 칼만 필터의 선형 최소평균제곱오차(Linear Minimum Mean Square Error, LMMSE) 직교성을 같은 구조로 증명한다.
  3. 조지프 형식이 양의 준정부호성(positive semidefiniteness, PSD)을 보존함을 이차형식으로 증명한다.
  4. 정확한 사후분포, LMMSE, 경험적 선형화(heuristic linearization)의 보장 수준을 비교한다.
  5. NIS가 크다는 이유만으로 \(R\)을 키우지 말고 편향(bias)·시간표시(timestamp)·좌표계(frame)·상관관계(correlation)를 먼저 점검한다.

# 칼만 필터는 한 번에 이해하지 않아도 된다

칼만 필터(Kalman filter)는 선형대수·확률·동역학이 합류하므로 처음부터 모든 유도를 밀어붙이면 어느 축에서 막혔는지 알기 어렵다. 아래 경로는 **작동 감각 → 계산 → 유도 → 증명**을 분리한다.

## 1. 직관 경로 — 약 70분

```text
평균·분산 → 상태와 측정 → 두 추정의 가중 결합 → 예측/보정
```

1. [평균과 공분산](../concepts/probability/mean-covariance.qmd)의 30초 핵심과 스칼라 분산을 읽는다.
2. [상태공간](../concepts/systems/state-space.qmd)의 등속도(constant-velocity) 예를 읽는다.
3. [칼만 필터의 스칼라 예](../concepts/estimation/kalman-filter.qmd#스칼라-최소-예)를 손으로 계산한다.

이 단계에서는 확률과정, 슈어 여원(Schur complement), 행렬미적분(matrix calculus)이 필요하지 않다.

## 2. 구현 경로 — 약 3시간

1. $F,H,Q,R,P$의 모양(shape)과 단위를 표로 작성한다.
2. [예측(prediction)](../concepts/estimation/kalman-filter.qmd#sec-kf-prediction)과 [보정(correction)](../concepts/estimation/kalman-filter.qmd#sec-kf-correction)을 한 단계 계산한다.
3. 역행렬을 직접 만들지 않고 선형계 풀이(`solve`)를 사용하며, 조지프 형식으로 공분산을 갱신한다.
4. 측정 결측(measurement dropout)에서는 보정을 건너뛰고 예측만 수행한다.
5. 혁신(innovation)과 정규화 혁신제곱(NIS)을 기록한다.
6. [Go2·L2 교육용 수치 사례](../concepts/estimation/kalman-filter.qmd#숫자로-한-번-보정하기-제품-측정값이-아닌-교육용-합성-예시)에서 교차공분산이 속도와 편향을 함께 고치는 과정을 확인한다.

## 3. 유도 경로 — 약 6시간

```text
공동 가우스 → 조건부 분포 → 칼만 이득
       └→ 투영/가중최소제곱 → 같은 보정
```

1. [가우스 조건부 분포(Gaussian conditioning)](../concepts/probability/gaussian-conditioning.qmd)를 유도한다.
2. [행렬 보정의 가우스 조건부 유도](../concepts/estimation/kalman-filter.qmd#행렬-보정을-가우스-조건부-분포로-유도한다)로 복귀한다.
3. [최소제곱](../concepts/linalg/least-squares.qmd)의 가중최소제곱과 직교성 관점을 읽는다.
4. 가우스 조건부 분포와 가중최소제곱이 같은 칼만 이득에 도달함을 설명한다.

## 4. 증명·진단 경로 — 9시간 이상

1. [직교투영](../concepts/linalg/orthogonal-projection.qmd)에서 잔차의 직교성을 증명한다.
2. 칼만 필터의 선형 최소평균제곱오차(Linear Minimum Mean Square Error, LMMSE) 직교성을 같은 구조로 증명한다.
3. 조지프 형식이 양의 준정부호성(positive semidefiniteness, PSD)을 보존함을 이차형식으로 증명한다.
4. 정확한 사후분포, LMMSE, 경험적 선형화(heuristic linearization)의 보장 수준을 비교한다.
5. NIS가 크다는 이유만으로 $R$을 키우지 말고 편향(bias)·시간표시(timestamp)·좌표계(frame)·상관관계(correlation)를 먼저 점검한다.

## 이 경로의 완료 기준

- [ ] 스칼라 칼만 이득을 불확실성 비율로 설명한다.
- [ ] 모든 행렬의 모양과 단위를 확인한다.
- [ ] 예측과 보정을 직접 유도한다.
- [ ] 가우스 모형에서의 정확성과 LMMSE 보장을 구분한다.
- [ ] $Q,R,P_0$를 물리적 잡음모형으로 설명한다.
- [ ] 혁신, NIS, 양의 준정부호성, 관측가능성(observability) 관련 실패를 분류한다.
