향후 확장 노드: 로봇·AI·역학·센서·구동기

세 핵심 장 이후 위키형 지식 아틀라스에 추가할 예정 개념의 안정 ID와 한 문장 범위

이 페이지는 목차 계약이며 본문이 아니다

현재 상세 집필 범위는 자코비안, 최대우도추정, 칼만 필터 세 장이다. 아래 항목은 위키 링크와 선수관계를 미리 안정화하기 위한 예정 노드(planned node)이며, 이번 단계에서는 안정 ID, 한 문장 정의, 첫 연결만 둔다.

표의 ←는 “왼쪽 노드를 이해할 때 오른쪽 기존 장이 직접 쓰인다”는 첫 연결이다. 상세 선수관계는 실제 장을 집필할 때 깊이별로 다시 감사한다.

로봇 기구학·제어·위치추정

안정 ID 예정 장 한 문장 범위 첫 연결
robot.forward_kinematics 순기구학 관절값에서 말단 위치·자세를 계산한다. ← 자코비안
robot.inverse_kinematics 역기구학 원하는 말단 자세를 만드는 관절값을 구한다. ← 자코비안, 최적화
robot.differential_kinematics 미분기구학 관절속도와 작업공간 속도를 국소적으로 연결한다. ← 자코비안
robot.manipulability 조작성 자코비안의 특잇값과 타원으로 방향별 운동·힘 능력을 평가한다. ← 자코비안, SVD
robot.trajectory_generation 궤적 생성 위치·속도·가속도 제한을 만족하는 시간 경로를 만든다. ← 수치선형대수, 최적화
robot.motion_planning 동작 계획 장애물과 동역학 제약 아래 시작 상태에서 목표 상태로 가는 경로를 찾는다. ← 최적화, 확률
robot.impedance_control 임피던스 제어 접촉 시 말단의 힘–변위 관계를 원하는 질량·감쇠·강성처럼 만든다. ← 자코비안, 상태추정
robot.force_control 힘 제어 접촉 힘·토크를 측정하고 원하는 렌치를 관절 명령으로 바꾼다. ← 자코비안, 제어
robot.operational_space_control 작업공간 제어 관절 동역학을 작업공간 힘·가속도 명령으로 바꾼다. ← 자코비안, 강체동역학
robot.state_estimation 로봇 상태추정 센서와 동역학으로 직접 측정하지 못하는 위치·속도·편향을 추정한다. ← MLE, 칼만 필터
robot.sensor_fusion 로봇 센서 융합 서로 다른 자료율·좌표계·불확실성을 가진 센서를 일관된 상태로 결합한다. ← 자코비안, MLE, 칼만 필터
robot.localization 위치추정 지도 또는 기준 좌표계에서 로봇 위치·자세와 불확실성을 추정한다. ← 칼만 필터, 입자 필터
robot.slam 동시적 위치추정 및 지도작성 알 수 없는 환경의 지도와 로봇 자세를 함께 추정한다. ← 자코비안, MLE, 칼만 필터
robot.calibration 로봇 보정 관절 영점·링크·도구·베이스 모수를 측정자료로 추정한다. ← 자코비안, MLE
robot.dynamics 로봇 동역학 다관절 로봇의 관성·코리올리·중력·접촉 항을 구성한다. ← 자코비안, 다물체계

역학·기구학

안정 ID 예정 장 한 문장 범위 첫 연결
mechanics.rigid_body_kinematics 강체 기구학 변형 없는 물체의 위치·자세·속도·가속도를 좌표계와 함께 기술한다. ← 자코비안, \(SE(3)\)
mechanics.rigid_body_dynamics 강체 동역학 힘·토크와 질량·관성에 따른 운동을 기술한다. ← 선형대수, 미분방정식
mechanics.lagrangian_dynamics 라그랑주 동역학 에너지와 일반화 좌표로 다관절 운동방정식을 유도한다. ← 행렬미적분, 최적화
mechanics.contact_friction 접촉·마찰 접촉 제약, 법선력, 마찰원뿔과 미끄럼을 모델링한다. ← 최적화, 역학
mechanics.multibody_system 다물체계 여러 링크·관절·접촉이 연결된 계의 운동을 효율적으로 계산한다. ← 강체동역학, 자코비안
mechanics.system_identification 시스템 식별 입력·출력 데이터에서 질량·마찰·모터 상수 같은 동역학 모수를 추정한다. ← MLE, 최적화

인공지능·로봇학습

안정 ID 예정 장 한 문장 범위 첫 연결
ai.supervised_learning 지도학습 입력–정답 자료에서 예측함수의 모수를 학습한다. ← 최소제곱, MLE
ai.neural_network 신경망 합성된 비선형 층과 역전파로 복잡한 함수를 근사한다. ← 행렬미적분, 최적화
ai.probabilistic_learning 확률적 학습 예측값뿐 아니라 불확실성과 잠재변수를 확률모형으로 학습한다. ← MLE, 베이즈 추론
ai.reinforcement_learning 강화학습 상태·행동·보상 상호작용에서 장기 의사결정 정책을 학습한다. ← 확률과정, 최적화
ai.robot_learning 로봇학습 시연·상호작용·센서 자료로 로봇의 인지와 행동을 학습한다. ← 상태추정, 제어, 강화학습
ai.differentiable_simulation 미분가능 시뮬레이션 시뮬레이터 출력의 미분을 이용해 모수·제어·설계를 최적화한다. ← 자코비안, 역학
ai.sim_to_real 시뮬레이션-현실 전이 시뮬레이터와 실제 로봇의 차이를 모델링하고 정책·모형을 이전한다. ← MLE, 시스템 식별

통계·필터 직접 확장

안정 ID 예정 장 한 문장 범위 첫 연결
statistics.fisher_information 피셔 정보 로그우도의 기대 곡률로 모수의 국소 정보량과 정밀도 한계를 기술한다. ← MLE
statistics.maximum_a_posteriori 최대사후확률추정 우도와 사전분포를 결합해 사후확률이 가장 큰 모수를 찾는다. ← MLE, 베이즈 법칙
statistics.generalized_linear_model 일반화선형모형 가우스 밖의 반응분포와 연결함수로 선형 예측자를 확장한다. ← MLE
signals.low_pass_filter 저역통과필터 낮은 주파수 성분을 통과시키고 높은 주파수 성분을 줄인다. ← 신호 및 시스템
estimation.extended_kalman_filter 확장 칼만 필터 비선형 동역학·측정을 자코비안으로 국소 선형화해 가우스 상태를 갱신한다. ← 자코비안, 칼만 필터
estimation.square_root_filter 제곱근 필터 공분산 대신 그 인수를 전파해 수치 안정성을 높인다. ← 수치선형대수, 칼만 필터
estimation.rts_smoother RTS 평활기 전방 칼만 필터와 후방 재귀로 미래 측정을 과거 상태에 반영한다. ← 칼만 필터
estimation.particle_filter 입자 필터 비선형·비가우스 상태분포를 가중 입자로 근사한다. ← 베이즈 필터, SMC

sensor.range_contact 거리·접촉 센서 초음파·적외선·스위치·촉각으로 근접과 접촉을 측정한다. ← 신호처리, 확률
sensor.calibration 센서 보정 편향·척도·축 불일치·외부 변환을 데이터로 추정하고 검증한다. ← 자코비안, MLE
sensor.time_sync 시간 동기화 여러 센서의 측정 시각·지연·클록 드리프트를 공통 시간축에 맞춘다. ← 칼만 필터

hardware.compute_module 연산 모듈 CPU·GPU·MCU의 계산량, 지연, 전력, 배치 제약을 비교한다. ← 알고리즘 구현
hardware.safety_io 안전 입출력 비상정지·인터록·제한 스위치·안전 상태 전이를 구성한다. ← 안전제어

확장 순서

  1. 세 핵심 장의 본문·링크·출력 검증을 끝낸다.
  2. 위 표의 안정 ID를 아틀라스 예정 노드 원본으로 승격한다.
  3. 독자가 가장 많이 따라간 역링크를 기준으로 다음 장을 고른다.
  4. 새 장을 만들 때만 깊이별 선수 체크와 실제 제품 근거를 추가한다.

상세 장이 생기기 전까지 이 페이지의 한 문장 범위를 넘어선 제품·알고리즘 주장은 추가하지 않는다.

선수관계를 미리 안정화하기 위한 **예정 노드(planned node)**이며, 이번 단계에서는 안정 ID,

실제 장을 집필할 때 깊이별로 다시 감사한다.

| `robot.calibration` | 로봇 보정 | 관절 영점·링크·도구·베이스 모수를 측정자료로 추정한다. | ← 자코비안, MLE |
| `robot.dynamics` | 로봇 동역학 | 다관절 로봇의 관성·코리올리·중력·접촉 항을 구성한다. | ← 자코비안, 다물체계 |
