# 스타일 프로파일

편집 이벤트(`events/*.jsonl`)의 주기 분석이 증류해 내는 감독자 취향·기준의
누적 문서다 (EDITING_SYSTEM_PLAN.md §7, Phase 3-2). **이 문서 자체는 규범이
아니다** — 여기서 숙성된 항목이 감독자 승인을 거쳐 `AUTHORING_MANUAL.md`
(최종 저장소)와 `style-rules.yml`(기계 검사)로 승격된다.

갱신 주체: 에이전트가 주기 분석으로 갱신안 PR을 만들고 감독자가 병합한다.
사람이 직접 편집해도 되지만, 근거 이벤트 링크가 없는 항목은 추가하지 않는다.
