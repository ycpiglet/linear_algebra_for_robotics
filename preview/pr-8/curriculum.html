전체 커리큘럼 지도

선형대수에서 제어·추정·SMC·MCMC까지 이어지는 비선형 학습 설계

먼저 결론: 순서가 아니라 네트워크다

아래 번호는 종이책을 위한 권장 참조 순서이지 수강 허가 조건이 아닙니다. PID처럼 바로 시작할 수 있는 주제는 먼저 읽고, 페이지의 선수 체크에서 막힌 개념만 되돌아갑니다. 한 개념도 직관 → 계산·적용 → 유도 → 증명·가정 → 가르치기의 숙달 상태를 따로 가집니다.

지금 알고 싶은 것 직관에 필요한 최소 선수 구현·유도에서 추가 증명 모드에서 추가
PID 오차, 음의 피드백 변화율·누적, sampling, saturation 대상 모델, 폐루프 안정성
최소제곱 벡터, \(Ax\)의 열 조합 직교투영, QR/SVD, 조건수 내적공간, convexity, rank
Kalman Filter 평균·공분산, 상태와 측정 선형 시스템, 상태공간, Gaussian 조건부 LMMSE 직교성, PSD, 확률 가정
Particle Filter Bayes 갱신, Monte Carlo IS, SIS, ESS, resampling 측도·적분, SLLN, 조건부기댓값
MCMC 확률분포와 조건부확률 Markov chain, 불변분포, detailed balance kernel, irreducibility, ergodic theorem
로봇 Jacobian 벡터미적분, 좌표계 matrix calculus, 선형화, SVD 미분사상과 rank theorem(선택)
센서 융합·로컬라이제이션 frame, noise, Bayes SE(3), KF/PF, 관측가능성 Lie group 확률·수렴론(선택)
힌트권장 사용법

목표 페이지의 30초 핵심을 먼저 읽고, 직관 체크를 통과하면 계속 갑니다. 계산에서 막히면 필요한 선수 하나만 우회하고 돌아가기 버튼으로 원래 문단에 복귀합니다.

A. 공학 수학의 공통 언어

A0. 표기·단위·계산 습관

  1. scalar, vector, matrix, tensor와 shape
  2. index, summation, transpose, inverse, pseudoinverse
  3. 물리 단위와 차원 분석
  4. 좌표 frame, active/passive transformation
  5. exact equality, approximation, heuristic, numerical error의 구분
  6. Python/NumPy 배열, broadcasting, 재현 가능한 난수

통과 기준: 식을 계산하기 전에 각 항의 shape·단위·frame을 표로 적을 수 있다.

A1. 미적분의 공학 직관

  1. 함수와 변화량
  2. 미분: 국소 선형화와 민감도
  3. 적분: 누적량과 면적
  4. Taylor 전개와 remainder의 의미
  5. 연속시간과 이산시간, sampling과 차분
  6. gradient, directional derivative, Jacobian의 예고편

바로 연결: PID의 I/D항, 상태방정식 이산화, optimization gradient.

## 먼저 결론: 순서가 아니라 네트워크다

아래 번호는 종이책을 위한 **권장 참조 순서**이지 수강 허가 조건이 아닙니다. PID처럼
바로 시작할 수 있는 주제는 먼저 읽고, 페이지의 선수 체크에서 막힌 개념만 되돌아갑니다.
한 개념도 `직관 → 계산·적용 → 유도 → 증명·가정 → 가르치기`의 숙달 상태를 따로 가집니다.

| 지금 알고 싶은 것 | 직관에 필요한 최소 선수 | 구현·유도에서 추가 | 증명 모드에서 추가 |
|---|---|---|---|
| PID | 오차, 음의 피드백 | 변화율·누적, sampling, saturation | 대상 모델, 폐루프 안정성 |
| 최소제곱 | 벡터, $Ax$의 열 조합 | 직교투영, QR/SVD, 조건수 | 내적공간, convexity, rank |
| Kalman Filter | 평균·공분산, 상태와 측정 | 선형 시스템, 상태공간, Gaussian 조건부 | LMMSE 직교성, PSD, 확률 가정 |
| Particle Filter | Bayes 갱신, Monte Carlo | IS, SIS, ESS, resampling | 측도·적분, SLLN, 조건부기댓값 |
| MCMC | 확률분포와 조건부확률 | Markov chain, 불변분포, detailed balance | kernel, irreducibility, ergodic theorem |
| 로봇 Jacobian | 벡터미적분, 좌표계 | matrix calculus, 선형화, SVD | 미분사상과 rank theorem(선택) |
| 센서 융합·로컬라이제이션 | frame, noise, Bayes | SE(3), KF/PF, 관측가능성 | Lie group 확률·수렴론(선택) |
