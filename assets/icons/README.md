# Icon rules

- 기본 grid: `24 × 24`, `viewBox="0 0 24 24"`
- 기본 stroke: `1.75`, round cap/join
- 의미를 담는 SVG에는 `<title>`을, 순수 장식에는 빈 `alt` 또는 `aria-hidden`을 사용한다.
- 하나의 아이콘만으로 상태를 구분하지 않고 label과 색을 함께 사용한다.
- 본문 block은 EPUB 호환성을 위해 `_content-blocks.scss`의 문자 icon과 heading을 기본으로 한다.
- 웹 전용 control은 Quarto에 포함된 Bootstrap Icons를 우선 재사용한다.
