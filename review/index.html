## 증명 해설서(Proof Companion)

이 판은 입자 필터(Particle filter)·순차몬테카를로(SMC)·마르코프연쇄 몬테카를로(MCMC)의 **가정, 보조정리, 유도, 완전증명**만 집중해서
읽는 독립 교재입니다. 직관이 필요하면 개념 장으로, 계산 검산이 필요하면 마지막 두
Python 실험으로 이동합니다.
:::

## 두 개의 완결 경로

- [입자 필터 경로](content/paths/particle-filter.qmd): 중요도표집(IS) 항등식 → 자기정규화 중요도표집(SNIS) 일관성 →
  순차중요도표집(SIS) 재귀 → 재표집 불편성 → 유한상태 고정 구간 입자 필터 일관성
- [MCMC 경로](content/paths/mcmc.qmd): 마르코프 연쇄 → 상세균형 → 메트로폴리스–헤이스팅스 불변성 →
  유한상태 총변동(TV) 수렴과 에르고딕 평균
- [전체 증명 지도](proofs.qmd): 각 명제의 범위와 의존관계를 한눈에 확인

::: {.callout-important title="증명 범위"}
현재 완결 정리는 유한 상태와 명시된 고정 horizon 범위입니다. 일반 상태공간의
파인먼–카츠(Feynman–Kac) 안정성, 해리스 재귀성(Harris recurrence), 일반 마르코프 핵(Markov kernel)의 에르고딕 정리(ergodic theorem)까지
증명했다고 확대 해석하지 않습니다.
:::
